# KeelungEat

### Requirements
-   [Node.js](https://nodejs.org)

```bash
sudo apt-get update
sudo apt-get install nodejs
sudo apt-get install npm
```

### Configuration
server.config.js
```javascript
const server_address = "http://localhost:5000"
```

### Developing
```bash
npm run-script dev
```

### Build
```bash
npm install
npm run-script build
```

### Run
```bash
npm start
```
